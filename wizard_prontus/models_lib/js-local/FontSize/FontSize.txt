
FontSize.class.js
------------------------------

Descripcion:
    Cambia el tamaño de la fuenta para los estilos que coincidan con cierto patrones
    como titular, bajada, cuerpo, etc. Véase la sección de configuración más abajo
    para cambiar dichos parámetros.

    Antiguamente se usaba un script que insertaba los íconos para cambiar el tamaño de la fuente
  sólo si el Browser era capaz. Ahora, como los navegadores obsoletos son cada vez más escasos,
  se opta por insertar los botones y enviar una alerta si el navegador es muy antiguo.

    OJO: Cambia el estilo de la primera hoja de estilos definida en la página
    Próximamente podría quedar obsoleto, por lo que la documentación es básica

Dependencias:
    Ninguna

Versión:
    ----------------------------------------------------------------------------
    2.0.0 - 11/11/2009 - CVI - Primera Versión coa Orientación a Objetos

Instalacion:
    Para instalar este script, sólo basta con agregarlo en el head, y luego
    para los botones que cambian los tamaños, se pone:
    FontSize.cambiaSize('mas') para agrandar la letra
    FontSize.cambiaSize('menos') para disminuir la letra

Configuración:
    Al comienzo del script están las configuraciones:
        // Tamaño actual En porcentaje
        size_actual: 100,
        // Arreglo con los tamaños a los que se podrá cambiar
        tamanos: [70,80,90,100,130,180,250],
        // patrones que se buscarán para cambiar el estilo
        estilos: ['titular','bajada','epigrafe','cuerpo'],
        // indice del arrglo de tamanos (3 = 4to elemento)
        size_actual_i: 3,
        // Mensaje cuando no se puede usar esta opcion
        msgError: 'Este Navegador no soporta aumento/disminución de tamaño de texto.',
        // En el caso de que se quiera procesar otro CSS y no el primero
        indice_hoja_estilos: 0,
    O bien, se puede hacer post carga, modificando los atributos. Ejemplo:
    $(document).ready(function() {
        FontSize.estilos = ['titular','bajada','epigrafe','cuerpo']; // Si se desean agregar nuevos estilos
        FontSize.indice_hoja_estilos = 1;  // Para trabajar sobre la segunda hoja de estilos.
    });
