Versión 3.0.3 - 25/04/2011 - CPN
----------------------------------------------------------------------------------------------------
+ Se agrega la macro macro_tags_relac.html para relacionar artículos por tags.
+ Se modifica la macro comun_relacionados.html, se incorpora la relación por tags.
  Los artículos relacionados que serán visualizados dependerá de lo siguiente:
  1º Si existe taxonomía manual se muestran los artículos relacionados por taxonomía manual.
  2º Si no existe taxonomía manual y existen tags para el artículo, se muestran los artículos relacionados por tags.
  3º Si no existe taxonomía manual y no existen tags para el artículo, y el artículo tiene una categoría (seccion, tema), se muestran los
     artículos relacionados por categoría.

Versión 3.0.4 - 05/05/2011 - CPN
----------------------------------------------------------------------------------------------------
+ Vistas.class.js, Se agrega la opción para redireccionar a una URL luego del seteo de la cookie. Ahora cuando se llame a la función js para el cambio de vista,
  puedo enviar de manera opcional la URL a la cual va a redireccionar, esto es un ejemplo practico cuando sitios web usan pre-home y queremos que luego de setear la cookie
  nos envié a la portada de inicio, pero en la vista antes escogida.

  <a href="#" onclick="Vistas.change('es', '/<prontus_xxx>/site/edic/base/port/inicio.html');">ESPAÑOL</a>
  <a href="#" onclick="Vistas.change('en', '/<prontus_xxx>/site/edic/base/port/inicio.html');">ENGLISH</a>
