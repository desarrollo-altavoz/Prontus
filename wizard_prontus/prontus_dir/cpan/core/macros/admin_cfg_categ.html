Variable Valor
TAXONOMIA_NIVELES
Indica el número de niveles de sección, tema y subtema a utilizar en el FID.
Atención: Dejando la variable TAXONOMIA_NIVELES en 0, se ha deshabilitado el uso de categorías en el sitio web, y por lo tanto, el resto de las variables de esta página no tendrán efecto.
NUM_RELAC_DEFAULT
Número de links hacia artículos relacionados que aparecerán junto a cada articulo.
TAXPORT_ARTXPAG
Número de artículos por página que aparecerán en las Portadas taxonómicas.
TAXPORT_MAXARTICS
Número máximo de artículos a considerar para construir las portadas taxonómicas. De esta manera se puede limitar su número por razones estéticas o de rendimiento del servidor.
Atención: La variable TAXPORT_MAXARTICS, ha superado el límite de seguridad. Este número se recomienda que sea menor a 1000, para no afectar el rendimiento de Prontus.
TAXPORT_ORDEN
Permitir definir el orden del listado de artículos en las portadas taxonómicas.
TAXPORT_TIPO_PAGINACION
Especifica el tipo de paginación para las portadas taxonómicas.
La misma que se usa desde el inicio de Prontus. El total de páginas es desplegado en pantalla.


También conocida como estilo Google, ya que fue popularizada por este buscador. Se muestran sólo X páginas hacia adelante y atrás. Habrá siempre un link a la primera y última página
TAXPORT_PAGCORTA_MAXPAGS
Sólo válido si se especifica la paginación corta. Indica la cantidad de páginas máximas que se mostrarán hacia adelante y hacia atrás. Este valor incluye la primera y última páginas
