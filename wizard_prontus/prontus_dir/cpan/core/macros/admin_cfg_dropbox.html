Variable Valor
DROPBOX_ACCESS_TOKEN
Access Token de la aplicación para poder interactuar con la API de DropBox.
DROPBOX_FILEXT_EXCLUDE
Listado de extensiones de archivos que serán excluidos del proceso de respaldo (separadas por coma, sin punto).
DROPBOX_CUSTOM_DIR
Directorios adicionales del artículo que se respaldarán. Estos se buscarán en el directorio /artic/<fecha_dir>/
Por defecto, solo se consideran los archivos que tengan en el nombre el TS del artículo.
