Seleccione el archivo que contiene los datos a importar:

El formato del archivo es el sgte:
  • Archivo xml con información de secciones, temas y subtemas.
  • Si se especifica id, el sistema lo utilizará. Si no, el sistema lo asignará automáticamente.
  • El formato de importación es el mismo de exportación.

IMPORTANTE: Al importar, las secciones, temas y subtemas que existan serán eliminados y reemplazados por lo que venga en el archivo a importar.
