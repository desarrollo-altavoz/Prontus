Modelo que permite crear sitios web limpios de gráfica, sin contenido publicado y sin información específica
Esta compuesto por:
  • Solo cuenta con la portada de Inicio (sin contenido, sin gráfica).

  • Cuatro tipo de artículo: Formulario (sin gráfica), General (sin gráfica), Link y Video (sin gráfica)

  • Portada de Categoría (sin gráfica)

  • Incluye set objetos javascript: Botones.class.js, FechaHora.class.js, Flash.class.js, FontSize.class.js, Formulario.class.js, Utiles.class.js, Vistas.class.js.

  • Incluye la librería jQuery versión 1.3.2
  • Macro subtitulos.
  • Botonera de herramientas (imprimir, enviar, maximizar y minimizar el tama√±o letra)
  • Macro compartir Facebook.
  • Sistema de comentarios.

Características del modelo:
  • Sitio con definición DOCTYPE XHTML 1.0 STRICT

  • Modelo trae implementado el sistema de comentarios en el artículo general.

  • Modelo incluye la macro de subtítulos

  • Modelo incluye cuadro de herramientas: enviar, imprimir, agrandar, achicar.

  • Modelo cuenta con CSS básicos.
Las portadas disponibles en el sitio son:
  • Inicio: Portada principal del sito no posee contenido en el body.

Las plantilla de portada Inicio solo posee la estructura general HTML.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>Prontus CMS</title>
</head>
<body>
    <!-- Prontus modelo vacio -->
</body>
</html>
Los artículos disponibles son:
  • General: Usado para mostrar artículos completos que pueden ser enlazados desde las portadas o visualizados por si solo. Solo posee en su interior el listado de marcas Prontus disponibles.

  • Formulario: Permite crear formularios para la interacción del cliente con el sitio, formularios de contactos, de consultas, contacto para compras, etc. El artículo esta operativo, pero no posee gráfica

  • Link: Permite crear enlaces (internos o externos) dentro de las portadas.
El modelo vacío incluye plantillas extras y que son comúnmente usadas en Prontus, Estas cuentan con la funcionalidad, pero no incluyen gráfica.
Las plantillas en el modelo son:
  • Portada de Categoría: cuentan con la estructura XHTML STRICT, las marcas Prontus y con la funcionalidad de la plantilla.

  • Plantillas de envío de artículos.

  • Plantilla Imprimir

  • Plantilla Mapa de portada

  • Plantilla Buscador Prontus

  • Plantilla RSS.
