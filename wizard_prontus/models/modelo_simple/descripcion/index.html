El modelo simple contiene una estructura simplificada a nivel de portadas, permitiendo una simplicidad y rapidez en la navegación del sitio.

Esta compuesto por:
  • Dos portadas: Inicio, Genérica (portada tipo).

  • Seis tipos de artículos: Banner, Formulario, General, Link, Logo, Rotulo.

  • Buscador Prontus


Características del modelo:
  • Diagramación visual sencilla y clara.

  • Codificación XHTML 1.0 según los estándares W3C.
Las portadas disponibles en el sitio son:
  • Inicio: Portada principal del sito. Agrupa noticias en distintas áreas y con distintos tamaños para darles mayor o menor relevancia.


  • Genérica: Portada tipo con distintas posiciones para presentar las noticias, ideal para noticias de un tema en concreto.
    La función de esta portada consiste en ser un molde para crear nuevas portadas dependiendo de las necesidades del sitio.
Los artículos disponibles son:
  • Banner: Permite crear información publicitaria he informativa, publicada y visualizada solo desde las portadas.

  • Formulario: Permite crear formularios para la interacción del cliente con el sitio, formularios de contactos, para consultas, compras, etc.

  • General: Usado para mostrar artículos completos que pueden ser enlazados desde las portadas o visualizados por si solo.

  • Link: Permite crear enlaces (internos o externos) dentro de las portadas.

  • Logo: Permite crear artículos que contengan imágenes corporativas.
