Modelo Prontus: Productos

  • Descripción
  • Portadas
  • Artículos
Enfocado a empresas que cuentan con una serie de productos y/o servicios, y que necesitan dar a conocer a sus clientes.
Para esto, el modelo cuenta con una serie de artículos y portadas especificas como listados de productos, galerías, integradores, etc.

El modelo esta compuesto por:
  • Cinco portadas: Inicio, Proyectos, Integradores, Productos, Portadas de Categoría.

  • Cinco tipos de artículos: General, Productos, Features, Proyectos, Formulario.

  • Buscador Prontus.

  • Mapa del sitio.

Algunas características del modelo son:
  • Diseños redondeados compatibles con Internet Explorer, Firefox, Chrome, etc.

  • Codificación XHTML 1.0 según los estándares W3C.

  • Portada principal posee un área especial para publicar banners gigantes con efectos slideshow jQuery.
Las portadas disponibles en el sitio son:
  • Proyectos: Muestra el portafolio de proyectos de la organización, se despliega el titular, una imagen y el link del proyecto terminado. En esta portada se publican artículos del tipo Proyecto.

  • Integradores: Muestra la lista de empresas y/o partners que usan los productos y/o servicios de la organización.

  • Productos: Muestra la lista de los productos y/o servicios proporcionados por la organización, visualizando en la portada una imagen, el título y la descripción del bien o servicio. En esta portada se publican artículos del tipo Producto.

  • Inicio: Portada principal del sito.

  • Portadas de Categoría: Muestra los artículos Prontus organizados por categorías (Sección, Tema, Subtema).

  • Buscador Prontus: Despliega los resultados de las búsquedas efectuadas dentro del sitio.

  • Mapa del sitio: Muestra la estructura de navegación del sitio web.
Los artículos disponibles son:
  • General: Usado para mostrar artículos completos, que pueden ser enlazados desde las portadas o visualizados por si solos.

  • Productos: Permite crear artículos con información de un producto y/o servicio, estos se publican en la portadilla _p_menu_productos.

  • Features: Permite crear artículos que son publicados en la portadilla _p_menu_productos y que lista 20 características con imagen, además de un cuerpo y archivos adjuntos.

  • Proyectos: Permite crear artículos con información de los proyectos realizados, imágenes, descargas y una cita como bajada del articulo.

  • Formulario: Permite crear formularios para la interacción del cliente con el sitio, formularios de contactos, formularios para consultas, compras, etc.
