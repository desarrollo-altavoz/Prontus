Paso 1 de 3: Datos Básicos

Este paso está destinado a recopilar la información básica para la configuración inicial del publicador. Serán solicitados datos elementales que serán utilizados permanentemente en el momento de operar el Administrador de contenidos Prontus.

Identificador del Sitio Prontus. Este dato se utilizará, entre otras cosas, para nombrar la carpeta que contendrá los archivos necesarios para el funcionamiento del Administrador de contenidos Prontus y aquellos que se creen a través de éste (ej: artículos, portadas, etc.); además del archivo de configuración interno del publicador (nombre_publicador.cfg).

Nombre del Sitio Prontus. Este dato se utilizará para desplegarlo en la etiqueta 'title' de las páginas del sitio, por ejemplo: 'MiSitio.cl', 'Mi sitio', etc.

Base de Datos MySQL. En este punto, debe indicar los datos de conexión a la base de datos que utilizará con Prontus. Optativamente, puede indicar una cuenta especial para que el Asistente intente crear la base de datos. En cualquier caso, antes de comenzar a utilizar Prontus, la base de datos deberá encontrarse ya creada.

Servidor SMTP. Indica el servidor SMTP a utilizar en el sitio Prontus, el cual puede corresponder a dirección IP o nombre del servidor, por ejemplo: mail.misitio.cl, 200.14.1.3.

Paso 2 de 3: Seleccionar Modelo Prontus

Los modelos Prontus consisten en estructuras o formatos de sitios web. Por ejemplo, se puede seleccionar un modelo cuya estructura consista en una cabecera (head), menú derecho con links y el centro utilizarlo para desplegar la información principal, como es el caso del sitio de Prontus: http://www.prontus.cl.

Paso 3 de 3: Confirmar Configuración

Antes de realizar la instalación del Administrador de contenidos Prontus, se debe confirmar la configuración. Para esto se debe revisar el resumen que se depliega en la página, el cual debe coincidir con las opciones seleccionadas en los pasos anteriores.

Reporte de Instalación

Informe sobre término exitoso de la instalación. Si la base de datos requerida ya está creada, el Administrador de contenidos Prontus se encuentra completamente operativo para comenzar a ser utilizado. Para esto, basta con presionar el link Panel de Control e ingresar las claves de acceso preestablecidas:
» usuario: admin
» contraseña: prontus
Además, se despliega la lista de los directorios y archivos generados en la instalación, los cuales pueden ser visitados haciendo click sobre cada elemento del árbol desplegado en la página.
